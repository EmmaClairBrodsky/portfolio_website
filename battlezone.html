BattleZone

A recreation of the arcade game BattleZone, featuring custom vertex and fragment shaders.

Overview

This project recreates the iconic wireframe aesthetic of the original BattleZone while leveraging modern graphics programming techniques. The custom renderer brings the classic tank combat gameplay to life with enhanced visual fidelity.

Technical Highlights

  • Implemented custom vertex and fragment shaders with WebGL for 3D wireframe rendering
  • Created dynamic 3D terrain with 100+ procedurally-generated obstacles and grid-based battlefield geometry
  • Designed organized and extensible architecture across 8 JavaScript modules for physics, AI, and UI systems
  • Autonomous enemy navigation with collision avoidance and player tracking
  • Added "sleeping enemies" feature: enemies with proximity-triggered state transitions (dormant to active)

Development

Culminating graphics project for CSC 461 - Computer Graphics. Developed in Javascript and WebGL and applied fundamental graphics techniques such as shader development, 3D transformations, and lighting.
