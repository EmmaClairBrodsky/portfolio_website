Chip's Challenge

An algorithmic solution to the classic Chip's Challenge puzzle game using A* pathfinding and intelligent search strategies.

Overview

This project tackles the computational challenge of solving Chip's Challenge levels through automated pathfinding. By implementing the A* algorithm with custom heuristics, the agent can navigate complex puzzle layouts efficiently.

Algorithm Features

  • Developed in Java using A* search
  • Swing-based visualizer for pathfinding execution
  • Solved 100% of provided maps under a 1000-step constraint

Implementation Details

Devloped an autonomous agent that uses pathfinding and planning to explore maps, collect items, and solve puzzles. Culminating project in CSC 411 - Intro to AI.
