3D Chess Set

A cohesive garden-themed 3D chess set.

Overview

This project showcases 3D modeling and design skills through a complete chess set inspired by garden elements. Each piece is carefully designed to represent botanical themes while maintaining clear visual distinction and hierarchy for gameplay.

Creation Process

The chess set was modeled using Blender, with careful attention to proportion, original inspiration, detail, and visual appeal for ADN 112 - Introduction to 3D Design.
